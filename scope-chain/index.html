      # 作用域链

      在变量对象 (opens new window)中提到，当查找变量的时候，会从当前上下文的变量对象中查找，如果没有找到，就会从父级（词法层面的父级）执行上下文的变量对象中查找，一直找到全局上下文的变量对象，也就是全局变量。这样由多个执行上下文的变量对象构成的链表就是作用域链
      在Javascript中，函数的作用域在函数定义的时候就决定了，这是因为函数有一个内部属性[[scope]]，当函数创建的时候，就会保存所有父变量对象到其中，也可以理解为[[scope]]就是所有父变量对象的层级链，但是注意：[[scope]]并不代表完整的作用域链
      举个例子：

      function foo() {
	function bar() {
 		...
  }
}
// 函数创建时，各自的[[scope]]为
  foo.[[scope]] = {
  	globalContext.VO
  }
  bar.[[scope]] = {
    fooContext.AO,
  	globalContext.VO
  }
